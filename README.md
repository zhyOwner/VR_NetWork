## 项目功能

主要设计的端口

* 监控端（服务器端）担任服务器，任务分发等功能
* 操作端 负责任务的执行
* 漫游端 只负责观看操作端的执行

功能：用户登录、监控端任务分发、实时监控在线人数、信息同步
